== Description ==

Emita boletos gratuitamente no seu Woocommerce com o Boleto Cora. Conta grátis, TED grátis Cartão VISA sem anuidade!

Esta é uma integração oficial da Cora, desenvolvida, mantida e suportada por Divox Tecnologia Ltda que lhe permite oferecer a opção de pagamento por boleto na finalização de cada pedido no Checkout do Woocommerce. 

Ao instalar o plugin, você ganha acesso a:

- Conta empresarial grátis, sem burocracia, livre de taxas ou mensalidades, com cartão VISA (livre de anuidade).
- Emissão gratuita de boletos, a menor taxa do mercado só é cobrada depois que receber, sem surpresas, sem pegadinhas.
- Gestão de Cobranças Grátis pelo App.
- TED grátis pelo App.
- Interface amigável com design moderno e intuitivo.
- Integração fácil, sem complicações, requer apenas WooCommerce.
- Suporte ostensivo e permanente.

ATENÇÃO: Somente após a compensação do boleto é cobrada a tarifa em sua conta Cora. Apenas R$ 2,00 por boleto compensado. 

IMPORTANTE: 
- A partir de 02/05/2023 está prevista a redução de tarifa de compensação para R$ 1,70 e cobrança de R$ 0,30 por cancelamento.
- Ocorre baixa automática com cobrança se o boleto não for pago 3 meses após a data de vencimento.
- Integração e emissões continuam gratuitas.

A instalação é extremamente simples: carregue o arquivo do plugin, vá para as configurações do WooCommerce e siga as instruções.

== Frequently Asked Questions ==

= Quanto eu pago para usar este plugin? =

Absolutamente nada, o plugin é gratuito para quem abre uma conta grátis na Cora. Você pagará apenas uma pequena taxa por cada boleto somente depois que ele é pago pelo seu cliente. A taxa vigente é informada no site https://cora.divox.com.br

= O que eu preciso para usar este plugin? =

Você precisa de um ambiente WordPress com o componente para comércio eletrônico WooCommerce, sua conta de empresa na Cora e de uma chave de licença que é gratuita. Basta você abrir sua conta gratuitamente (sem taxas) e solicitar a chave de licença gratuita no site do plugin: https://cora.divox.com.br

= Precisa ter CNPJ? =

Sim. Cora opera apenas com pessoas jurídicas. Mas aceita empresas MEI.

= Não estou conseguindo autorizar a integração =

A integração funciona em 3 etapas. Você deve clicar em SALVAR antes de seguir para a próxima configuração.
